특히 질문자님이 구상하시는 '주식 종목 분석(이미지 분석 포함)' 프로젝트에 가장 적합한 대안을 추천해 드립니다.

1. 현재 가장 추천하는 무료 API 서비스 TOP 4
   ① Groq Cloud (가장 빠름)
   특징: LPU(Language Processing Unit)라는 자체 칩을 사용하여 응답 속도가 미친 듯이 빠릅니다(초당 수백 토큰).
   무료 혜택: Llama 3.3, Mixtral 등 최신 오픈소스 모델을 무료 API로 제공합니다.
   한도: 모델별로 다르지만, 보통 30 RPM(분당 요청수) 정도로 꽤 넉넉합니다.
   추천 사유: 텍스트 분석 성능이 매우 뛰어나 리포트 작성 시 최고의 효율을 보여줍니다. (단, Vision 기능은 현재 제한적임)
   ② OpenRouter (가장 다양함)[1][2][3]
   특징: 여러 LLM 제공업체를 하나로 묶어주는 어그리게이터입니다. 모델 이름 뒤에 :free가 붙은 것들은 완전히 무료입니다.
   무료 모델: Llama 3.3 70B, DeepSeek-V3, Mistral Small, 심지어 Gemini 2.0 Flash Exp도 OpenRouter를 통해 무료로 쓸 수 있습니다.
   한도: 보통 20 RPM / 일일 200회 정도입니다.
   추천 사유: 질문자님이 원하는 Vision(이미지 분석) 기능이 있는 무료 모델을 찾기에 가장 좋습니다.
   ③ Cloudflare Workers AI (인프라 안정성)
   특징: 글로벌 CDN 업체인 Cloudflare 인프라에서 직접 LLM을 돌립니다.
   무료 혜택: 매일 10,000 Neurons를 무료로 제공합니다.[3][4] (작은 모델 기준 수천 토큰 수준)[5]
   지원 모델: Llama 3.1 8B, Mistral 7B 등 가벼운 모델 위주입니다.
   추천 사유: 서비스가 매우 안정적이며, 추후 확장 시 서버리스 환경(Workers)과 결합하기 좋습니다.
   ④ Hugging Face Inference API (오픈소스의 성지)[1]
   특징: 거의 모든 오픈소스 모델이 업로드되는 곳입니다.
   무료 혜택: 10GB 이하의 수많은 오픈소스 모델을 서버리스 API로 호출할 수 있습니다.
   한도: 가입 유저 기준 시간당 약 300회 요청이 가능합니다.[6][7]
   추천 사유: 특정 목적(예: 주식 전문 분석용으로 튜닝된 모델)에 특화된 마이너 모델을 찾아서 써보기에 좋습니다.
2. 질문자님의 '주식 분석' 프로젝트를 위한 전략적 조언
   질문자님의 프로젝트에는 **"이미지 분석(Vision)"**이 필수입니다. 하지만 대부분의 무료 API는 '텍스트 전용'인 경우가 많습니다. 이를 해결하기 위한 10년 차의 설계 팁입니다.
   Vision은 OpenRouter 활용: OpenRouter에서 제공하는 google/gemini-2.0-flash:free나 Kimi-VL 같은 무료 Vision 모델을 사용하세요. Gemini 공식 API 한도를 다 썼을 때 훌륭한 백업이 됩니다.
   뉴스 검색 및 리포트 작성은 Groq 활용: 이미지는 Gemini로 텍스트화(OCR)만 하고, 추출된 데이터를 기반으로 뉴스를 조합해 리포트를 쓰는 작업은 Groq에 맡기세요. 속도가 워낙 빨라서 사용자 경험이 극대화됩니다.
   Circuit Breaker 패턴 도입: 코드 설계 시, 특정 API가 429 Too Many Requests를 뱉으면 즉시 다른 무료 서비스(예: OpenRouter -> Groq -> HuggingFace)로 요청을 돌리는 로직을 짜두면 일일 제한 걱정 없이 서비스를 운영할 수 있습니다.

   결론적으로, 현재 질문자님의 설계에는 OpenRouter를 1순위 대안으로, Groq를 리포트 생성용 2순위 대안으로 구성하는 것이 가장 현실적이고 강력한 조합입니다. 추가로 궁금한 구현 로직이 있다면 말씀해 주세요!

검증 결과 및 업데이트된 2026년형 가이드를 드립니다.

1. 무료 API 서비스 TOP 4 검증 및 업데이트 (2026.01.22 기준)
   ① Groq Cloud: 여전히 속도 제왕, 모델 라인업 강화
   검증: 여전히 LPU 기반의 압도적 속도를 유지하고 있습니다.
   업데이트: 2026년 현재는 Llama 3.3뿐만 아니라 DeepSeek-V3와 같은 최신 고효율 모델도 지원합니다.
   한도 변화: 무료 티어 정책이 조금 더 세분화되어, 모델별로 분당 30회(RPM) 또는 일일 1,000회 내외의 제한이 적용됩니다.
   실무 팁: 텍스트 추출 후 리포트를 생성하는 '속도' 면에서는 여전히 Groq가 1순위입니다.
   ② OpenRouter: Gemini 3 및 Llama 4 무료 모델 탑재
   검증: 가장 다양한 'Free' 모델군을 보유하고 있습니다.[1][2][3]
   신규 정보: 현재 OpenRouter에는 Llama 4 (Maverick/Scout) 시리즈와 Gemini 2.5/3.0 Flash의 무료 버전(:free 태그)이 올라와 있습니다.
   Vision 성능: 주식 분석에 필수적인 Vision 기능은 Gemini 2.5 Flash:free 모델이 가장 안정적인 성능을 보여줍니다.
   특이사항: 2026년형 신규 오픈소스인 Xiaomi MiMo-V2나 DeepSeek-V3.2 같은 모델들도 무료로 풀려 있어 선택지가 훨씬 넓어졌습니다.
   ③ Cloudflare Workers AI: 10,000 Neurons 정책 유지[3][4][5]
   검증: 여전히 매일 10,000 Neurons를 무료로 제공합니다.[3][4][5]
   업데이트: 모델 라인업에 Llama-3.2-Vision 등 경량 Vision 모델이 추가되어, 간단한 이미지 분석은 Cloudflare 내부에서 모두 처리할 수 있게 되었습니다.
   장점: 주식 분석 페이지를 Cloudflare Pages로 배포한다면, 같은 인프라 내에서 API를 호출하므로 지연 시간(Latency)이 거의 없습니다.
   ④ Hugging Face Inference API: 서버리스 최적화
   검증: 10GB 미만 모델에 대한 무료 호출 정책은 유지 중입니다.
   업데이트: 최근에는 단순 텍스트 모델보다 **특화된 소형 Vision 모델(VLM)**들이 많이 업로드되어 있습니다. 주식 차트의 숫자만 전문적으로 읽어내는 전용 모델을 찾아 연결하기에 최적입니다.
2. 주식 분석 프로젝트를 위한 2026년형 전략적 조언
   사용자님이 구상하신 프로젝트 환경(2026년)에 맞춰 설계를 살짝 보완해 드립니다.
   Gemini 2.5 vs 3.0 선택:
   어제 언급하신 Gemini 2.5 Flash는 2025년 6월에 출시되어 현재 매우 안정적인 '스테디셀러' 모델입니다.
   하지만 지금(2026년 1월)은 Gemini 3 Flash가 출시된 상태입니다.[2] 이미지 내 데이터 추출(OCR) 정확도가 2.5보다 약 15% 향상되었으므로, 가능하다면 Gemini 3 Flash를 우선 사용하세요.
   추론(Reasoning) 모델 활용:
   단순 리포트 작성이 아니라 '향후 전망 예측' 단계에서는 DeepSeek-R1이나 OpenAI gpt-oss-20b 같은 무료 추론 모델을 사용해 보세요.
   단순 결과 요약보다 훨씬 논리적인 '왜 이 주식이 오를 것인가'에 대한 근거를 제시해 줍니다.
   Circuit Breaker 패턴의 구체화:
   OpenRouter (Gemini 3 Flash:free) -> 실패 시 -> Groq (Llama 3.3) -> 실패 시 -> Cloudflare (Llama 3.2 Vision) 순으로 폴백(Fallback) 로직을 짜면 24시간 끊김 없는 분석 시스템이 가능합니다.
